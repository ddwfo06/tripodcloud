## 线路是核心，CN2 GIA + 联通 + 移动三网直连

TripodCloud 的所有 VPS 产品，走的都是 **CN2 GIA + 联通骨干 + 移动 CMI 三网直连**，机房在美国西海岸加州圣何塞（San Jose）。

对于国内用户来说，三网直连意味着：

- **电信用户**：走 CN2 GIA（AS4809），这是电信最高端的国际线路，延迟低、绕路少
- **联通用户**：直连联通骨干网（AS4837）
- **移动用户**：走移动 CMI 线路

有测评数据显示，用联通 50M 宽带跑这个线路，跑 1080P 视频非常流畅，延迟表现比普通 163 骨干低得多。圣何塞机房的 Speedtest 测速，上传能跑到 130+ Mbps，下载接近 800 Mbps，与产品标注的 1Gbps 端口相符。
